README.MD

I would like to thank the Tensorflow tutorials and Github user nlintz for parts of the codes for this workshop.

Ubuntu/Linux and Mac users can use the scripts below to install tensorflow. 

Windows users can run TensorFlow on https://codeenv.com/env/codeenv/14/tensor-flow/
	a) Hit the launch button on the top right corner (blue square with rocket inside)
	b) once environment runs, write on the terminal ipython

File Directory

1) Machine Learning.pptx -> Slide Presentation of the Workshop
2) Installation
	a) install_tensorflow_linux.sh -> script to install tensorflow on Ubuntu/Linux
	b) install_tensorflow_mac.sh -> script to install tensorflow on mac
	c) WIndows
		I) VirtualBox .exe
		II) VirtualBox Extensions
		III) Ubuntu 14.04 iso file
4) basics 
	a) var_create_save -> Show how to create, save, and load variables
	b) arith -> Common arithmetic operations
	
5) models
	a) linear_regression.py -> Simple Implementation of a linear_regression model
	b) logistic_regression.py -> Implementation of Logistic Regression model
	c) convolutional_net.py -> Implementation of Convolutional Network
